# Create Theme Json

File stucture:

```
theme-json
│   custom-templates.json
│   template-parts.json
│
└───settings
│   │   border.json
│   │   color.json
│   │   custom.json
│   │   layout.json
│   │   spacing.json
│   │   typography.json
│   │
│   └───blocks
│       └───core
│           │   paragraph.json
│           │   heading.json
│           │   ...
│
└───styles
    │   border.json
    │   color.json
    │   elements.json
    │   spacing.json
    │   typography.json
    │
    └───blocks
        └───core
            │   paragraph.json
            │   heading.json
            │   ...
```
