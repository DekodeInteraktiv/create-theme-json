# Create Theme Json

File stucture:

```
theme-json
│
└───settings
│   │   layout.json
│   │   color.json
│   │   typography.json
│   │
│   └───blocks
│       └───core
│           │   paragraph.json
│           │   cover.json
│           │   ...
│
└───styles
    │   color.json
    │   typography.json
    │   elements.json
    │
    └───blocks
        └───core
            │   paragraph.json
            │   cover.json
            │   ...
```

Supported file formats: `json`, `yml`
